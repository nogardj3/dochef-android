## How to use

1. 서버 구동 - https://github.com/nogardj3/server_nodejs 참조

2. Firebase Project 등록 및 API 연결

3. local.properties
   admob_app_id = ""
   admob_banner_id = ""

4. app/google-services.json 파일 추가

5. strings.yaml에 서버 urls 추가
